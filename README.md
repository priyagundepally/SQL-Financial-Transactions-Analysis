# SQL Project: Financial Transactions Dataset Analytics & Insights
# Source: Kaggle https://www.kaggle.com/datasets/computingvictor/transactions-fraud-datasets
Project provides insights into spending analysis by state category and brand, to merchants, banks and card networks for better organization of the resources. 
Transaction error analysis will help banks to provide overdraft facilities to customers with strong credit scores.  
The financial stability index helps banks to decide which states are more suitable for business expansion. 
Cards table has a column  "is_card_on_dark_web". Unfortunately dataset had his value as FALSE, if we had this data – we could better analyze how fraudulent transactions take place.
